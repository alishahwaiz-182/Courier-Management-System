#  Courier Management System

A full-stack courier management system built with PHP and MySQL featuring 
three separate panels for Admin, Agent, and User.

##  Features

###  Admin Panel
- Dashboard with overview
- Track orders
- Create & manage agents
- Manage customers
- Add & view couriers
- Reports
- Logout

###  Agent Panel
- Dashboard
- Add courier
- View couriers
- Reports
- Logout

###  User Panel
- Track order
- Logout

##  Built With
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

##  How to Run
1. Install XAMPP
2. Clone this repo in `htdocs` folder
3. Import database from `db` folder in phpMyAdmin
4. Open browser and go to `localhost/Courier-Management-System`
